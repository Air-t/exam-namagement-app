Info

Exam application allows user with owner permisions to create test template. All CRUD operations are available to manage exams. Owner can prepare exam, which may be composed of two types of questions - Close and Open one. Based on user anser, questions are evaluated as follows: Close questions are evaluated instantly once exam is passed. Close questions may are multi choice. Score is assigned to user who chooses all valid choices for a single question. Partially valid answers are evaluated as 0 points of all available within question. Open questions need owner action to be evaluated. Owner is responsible to invirte students via email. Clients need to be signed in with the same email address as one sent with invitation. Once is done, students may take one try to pass the exam. Exams has their own time within they need to be finished. Countdown timer begins ticking when client starts the exam. User may finish the exam, clicking finish button, or exam will automaticly finish once time is up. Based on given user answers, exam is evaluated and simple statistics are prepared.
